# Development Practices and Environmental Setup

Commands to compile and run code on `./src` folder.

## Compile
```bash
make compile dir=DIR file=FILE
```

## Run
```bash
make compile dir=DIR file=FILE
```

# Compile and Run
```bash
make compile-run dir=DIR file=FILE
```

Examples
```bash
make compile-run dir=misc file=hellow
```
